The Grid components provide a 12 column layout grid for organizing content for all device sizes. The grid consists of sky-row and sky-column tags. Every column must be contained in a row. Each column can have multiple properties specified in order to handle different behavior for each screen size. Below the width of 768px (small screen) all elements will be displayed in a single column. Reverses the display order for the child columns when set to true. The number of columns (1-12) the element will use on a small screen (768-991px). The number of columns (1-12) the element will use on a medium screen (992-1199px). The number of columns (1-12) the element will use on a large screen (1200px+).
