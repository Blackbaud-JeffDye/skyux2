The Grid components provide a 12 column layout grid for organizing content for all device sizes. The grid consists of sky-row and sky-column tags. Every column must be contained in a row.

Sky Row properties

  • reverseColumnOrder — (Optional) Reverses the display order for the child columns when set to true. (Default: false)

Sky Column properties

  • screenSmall — The number of columns (1-12) the element will use on a small screen (768px).
  • screenMedium — The number of columns (1-12) the element will use on a medium screen (992px).
  • screenLarge — The number of columns (1-12) the element will use a large screen (1200px).

To handle different behavior for multiple screen sizes, you can specify more than one property on each column. For example: <sky-column screenSmall="8" screenMedium="9" screenLarge="10">. This is helpful if you want to more carefully control the breakpoints.
