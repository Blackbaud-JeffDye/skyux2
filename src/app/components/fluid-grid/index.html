The fluid grid component provides a 12-column layout to organize responsive content for all device sizes. The grid consists of sky-row and sky-column tags. Each column must be contained in a row, and can specify behavior for different screen sizes. On extra small screens (less than 768px) the grid displays all elements in a single column. Indicates whether to reverse the display order for columns within this row. Specifies the number of columns (1-12) for the element to use on a small screen (768-991px). Specifies the number of columns (1-12) for the element to use on a medium screen (992-1199px). Specifies the number of columns (1-12) for the element to use on a large screen (greater than 1200px).
